# rails-stimulus-sass

Starter SaaS preconfigured boilerplate with Bootstrap 5, Devise, Notify, Annaouncements, Madmin, Stimulus JS and Action Cable.

Also can be used with React/Vue via  $rails webpacker:install:react (or vue)
